Sharkord plugins let you extend server behavior with custom logic such as automations, integrations, and commands. **This is still an experimental feature, major changes are expected as we iterate on the design and API.**

<Alert type="attention" title="Security Warning">
  Before installing plugins, please read the [Security](./security) page to
  understand the potential risks.
</Alert>

### Plugin SDK

Use the official SDK for type-safe plugin development: [packages/plugin-sdk](https://github.com/Sharkord/sharkord/tree/development/packages/plugin-sdk).
