This guide walks through creating a minimal Sharkord plugin.

### 1) Create a Plugin Directory

Initialize a new plugin:

```bash
bun create github.com/Sharkord/plugin-example
```

### 2) Link the needed packages

To do this, first clone the Sharkord repository:

```bash
git clone https://github.com/Sharkord/sharkord.git
```

Install dependencies and link the correct packages:

```bash
cd sharkord

bun install

cd ./packages/plugin-sdk
bun link

cd ../ui
bun link
```

Now exit the Sharkord repo and clone another repo: the plugin builder, which is a template for building plugins with the correct configuration and build scripts:

```bash
git clone https://github.com/Sharkord/plugin-builder.git
```

Then install dependencies and link the package:

```bash
cd plugin-builder

bun install

bun link
```

Then, in your plugin directory, link the SDK and UI:

```bash
cd plugin-example

bun link @sharkord/plugin-sdk
bun link @sharkord/ui
bun link @sharkord/plugin-builder
```

### 3) Change the manifest file

Go to `manifest.json` and edit the information to match your plugin. The `id` field is important, it should be a unique identifier for your plugin. It should be all lowercase. Example:

```json
{
  "id": "plugin-example",
  "name": "Plugin Example",
  "author": "Me",
  "description": "This a Sharkord plugin example.",
  "homepage": "https://some-page.com",
  "logo": "https://placehold.co/100x100/white/black",
  "sdkVersion": 1,
  "version": "0.0.1"
}
```

### 4) Build

Your plugin needs to be bundled before it can be used. There's a build script already set up for you, so you can just run:

```bash
bun run build
```

And your plugin will be bundled into the `dist` folder, which you can then install into your Sharkord instance (see [Installation](./installation) for more details).
