After compiling your plugin, install it into your Sharkord data directory.

<Alert type="attention" title="Security Warning">
  Before installing and using plugins, please read the [Security](./security)
  page to understand the potential risks.
</Alert>

### Via Marketplace

The easiest way to install plugins is through the built-in marketplace. This allows you to browse and install plugins with a single click, without needing to manually manage plugin files.

The marketplace lives in [https://github.com/Sharkord/plugins](https://github.com/Sharkord/plugins) and is open for anyone to publish their plugins. If you have developed a plugin and want to share it with the community, you can submit it to the marketplace.

### Manual Installation

1. Open your Sharkord data directory (see [here](../data-dir) for more info).
2. Create a `plugins` folder if it does not exist.
3. Create a folder for your plugin, for example `my-plugin`.
4. Copy your bundled plugin (for example, `dist/my-plugin`) into that folder.
5. Enable the plugin in Sharkord settings under **Plugins**.
6. Go to the plugins tab and the plugin should be listed there. Click **Enable** to activate it.
